# ☕ Home Brew — Coffee Shop Landing Page

> A promotional landing page for a fictional coffee shop, built as part of the **shiftUP Digital** portfolio by **Elio Mondlane**.

---

## 📋 About the Project

**Home Brew** is a single-page landing page designed for a coffee shop loyalty campaign. The page promotes a new member offer — a free drink on first visit — and is built to feel warm, inviting, and on-brand.

This project demonstrates proficiency in semantic HTML5, modern CSS architecture using custom properties, and vanilla JavaScript for interactivity.

---

## ✨ Features

- **Dark / Light Mode Toggle** — persists across sessions using `localStorage`
- **Fluid Typography** — font sizes scale smoothly across all screen sizes using `clamp()`
- **CSS Custom Properties** — full design token system for colors, spacing, and typography
- **Semantic HTML5** — proper use of `<header>`, `<main>`, `<section>`, `<article>`, and `<footer>`
- **Dynamic Copyright Year** — auto-updates via JavaScript; no manual maintenance needed
- **Google Fonts Integration** — Abril Fatface, Inter, and Poppins
- **Responsive Layout** — flexbox-based layout that adapts to any viewport
- **Accessible Markup** — meaningful meta tags, `lang` attribute, and `<time>` element

## 🎨 Design Decisions

- **Color Palette** — warm cream, caramel, and deep brown tones to evoke the feel of a coffee shop
- **Abril Fatface** — used for headings and brand elements to give a bold, editorial feel
- **Inter** — clean and readable for body text and UI labels
- **Dark Mode** — inverts to near-black backgrounds with the same warm accent tones, keeping the brand intact in both modes
- **CSS Variable Architecture** — all colors, spacing, and type scales are tokenised so the design system can be extended or rebranded with minimal changes
